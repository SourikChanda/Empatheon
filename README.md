# PTSD Support Chatbot

This is a simple NLP-powered chatbot that helps users by responding to PTSD-related concerns.

## Features
- NLP with sentence-transformer models
- FAISS for fast semantic search
- Streamlit frontend
- Fine-tuned for small datasets

## Run Locally

```bash
git clone https://github.com/yourusername/ptsd-chatbot.git
cd ptsd-chatbot
pip install -r requirements.txt
streamlit run app.py
